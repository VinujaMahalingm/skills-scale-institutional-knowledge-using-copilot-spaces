# OctoAcme Project Management Docs

This folder contains centralized documentation for OctoAcme’s project management processes, workflows, and standards.

## Project Management Process Overview

OctoAcme uses a structured but flexible project management approach focused on transparency, shared ownership, and continuous improvement. Documentation is treated as a living artifact and updated through issues and pull requests.

## Roles and Responsibilities

- **Product Owner** – Defines scope and priorities.
- **Project Manager** – Plans and tracks execution.
- **Engineering Lead** – Ensures technical quality.
- **Contributors** – Implement features and improve documentation.
